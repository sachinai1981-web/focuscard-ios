# Focus Card

An iPhone-first planner based on your paper card workflow:

GitHub repo: [sachinai1981-web/focuscard-ios](https://github.com/sachinai1981-web/focuscard-ios)

- tasks on the left
- actual time used on the right rail
- a larger color palette for time blocks
- start marker as a circle
- finish marker as a triangle

## What is built

- Add a task
- Edit a task
- Delete a task
- Pick a highlight color
- Choose start and end time in 15-minute steps
- See total tracked time
- Save tasks locally on the device

## Run it on your iPhone

1. Install `Expo Go` from the App Store on your iPhone.
2. In the project folder, run:

```bash
npm start
```

3. Make sure your Mac and iPhone are on the same Wi-Fi network.
4. Scan the QR code with the iPhone camera or from inside Expo Go.
